## Vestra

### Still a WIP

Vestra is a web application that does sentiment analysis on dozens of articles by taking live data through webscraping, according to the top 500 stocks, and then using the sentiment analysis to determine if the stock is a good investment utilising scheduled cron jobs.

## Features

- Interactable Visualization of the predicted stock prices with lower and upper bounds
- Ranking of the top 100 stocks based on the sentiment analysis
- Stock Prediction News
- Price Prediction per stock
